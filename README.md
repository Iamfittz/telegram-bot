# Telegram Bot on Go 

Simple Telegram bot built using **Go**, **Cobra**, and **Telebot**.

##  How to run

1. Install Go 1.25+
2. Clone the repo:
   ```bash
   git clone https://github.com/iamfittz/telegram-bot.git
   cd telegram-bot
3. Set your Telegram token:
   ```bash
   setx TELE_TOKEN "YOUR_TELEGRAM_TOKEN"
4. Run the bot:
   ```bash
   go run main.go
